Key Features

  • VB6-like properties: Text, Name, Enabled, Visible, BackColor, ForeColor, etc.
  • Event system: Click, Change, Focus, Blur, MouseEnter, MouseLeave, KeyPress
  • State management: normal, hover, active, disabled, focused
  • JSON serialization for saving/loading layouts
  • Instance mode p5.js - each component has its own canvas
  • Standard methods: getValue(), setValue(), enable(), disable(), show(), hide()
  • Fully customizable appearance and behavior

Usage Example

// Create a button
const btn = new P5Button({
  text: 'Click me',
  width: 100,
  height: 30,
  left: 10,
  top: 10,
  backColor: '#E0E0E0',
  foreColor: '#000000'
});

// Listen to events
btn.on('click', () => {
  console.log('Button clicked!');
});

// Change properties
btn.text = 'New Text';
btn.enabled = false;

// Serialize to JSON
const json = btn.toJSON();
console.log(json);
